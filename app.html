Academic demonstration. RE Underwriter is an interactive real-estate acquisition screening application I developed in collaboration with AI. It uses sample Northern Utah properties to demonstrate deal comparison, underwriting metrics, risk identification, renovation analysis, and investment recommendations. The listing-import workflow is simulated for this public, serverless version — it fills in a canned example so you can see the flow, and it is not performing live AI extraction. There is no login and no saved database, so a refresh resets everything.

How the full app differs: the version I use privately runs on a server and does perform live listing import — fetching a real listing from a pasted URL or screenshot, extracting the details with AI, downloading the photos, and saving deals to a database so I can revisit and update them over time. This public demo keeps all of that interactive on the front end, with the live/private pieces simulated. Everything else works: click a deal for its snapshot charts, tick boxes to compare, and open a deal for the full underwriting view.

Quick-start a deal from a listing · simulated in this demo

In the full app, you paste a listing URL or text (or drop screenshots) and AI fills in the intake fields and downloads the photos automatically — for UtahRealEstate, MLS, KSL, and FSBO listings. In this public demo this is simulated: submitting loads a canned sample listing so you can see the end-to-end workflow without a server or live AI.
